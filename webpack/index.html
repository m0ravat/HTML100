Webpack bundling

First run npm init Then, npm install webpack --save-dev then npm install --save-dev babel-loader @babel/core @babel/preset-env then npm install style-loader css-loader --save-dev npm install sass sass-loader --save-dev save-dev is used since these are used to build the app before it gets put in production, not used during app run so save dev is best for this. Express/react are run during prod to load pages so NOT save-dev Then npx webpack to run
